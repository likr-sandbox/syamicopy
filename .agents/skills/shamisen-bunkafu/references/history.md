# 文化譜の歴史的背景と普及の過程

## 三味線音楽の伝承における記譜法以前の世界

日本の伝統音楽、とりわけ三味線音楽のルーツは、中国の弦楽器「三弦」が沖縄に伝播して独自の進化を遂げた「三線（さんしん）」にある。
それが江戸時代にかけて北前船などの海運ルートを通じて日本本土へと伝わり、庶民の芸能とともに多様な発展を遂げてきた。

長らくこの音楽は、**「口伝（くでん）」や「手ほどき」** といった、師匠から弟子への直接的な対面伝授を前提として発展してきた。
その中で用いられてきたのが **「口三味線（くちじゃみせん）」** と呼ばれる、擬音語を用いた旋律の記憶法である。

### 口三味線の具体例

| 擬音 | 意味 |
|------|------|
| 「ドン」「ト」 | 一の糸の開放弦を弾く音 |
| 「ロン」 | 撥ですくい上げる音 |
| 「チン」「ツン」 | 高い音 |

これらを組み合わせて「チンチリリン」「トテウチン」のように、音高と音色、奏法を同時に言語化して伝達していた。
しかし、口頭伝承に依存するため個人の記憶や解釈による変動が避けられず、初心者にとって習得のハードルが極めて高いという課題を抱えていた。

## 大正期における邦楽の近代化と文化譜の誕生

### 考案の経緯

文化譜は、**大正11年（1922年）7月** に、長唄の四世家元・**杵家弥七（きねや やしち）** によって考案・創案された。

大正時代は、西洋音楽の教育体系が日本の学校教育に深く根付き始めた時期であり、邦楽界においても「再現可能な視覚的記録」へのニーズが急速に高まっていた。当時は三味線音楽を五線譜にそのまま翻訳して出版する試みもあったが、和楽器特有の演奏感覚と西洋音楽の五線譜とは認知的な齟齬が大きかった。

四世杵家弥七は、三味線という楽器の特性を五線譜に無理に当てはめるのではなく、**楽器の物理的な構造に直接対応する「直感的な操作指示書」** としての記譜法を追求した。その結果生み出されたのが、三味線の三本の弦と左手で押さえる「勘所（かんどころ）」を視覚的にマッピングした文化譜である。

### 「赤譜」の通称

表紙に赤い紙が用いられることが多かったことから、現在でも通称 **「赤譜（あかふ）」** と呼ばれ、広く親しまれている。

## 出版インフラの整備と市販化を通じた波及

### 出版体制の変遷

文化譜の普及には、楽譜の出版・流通インフラの整備が不可欠であった。

| 時期 | 出来事 |
|------|--------|
| 大正11年（1922年） | 四世杵家弥七が **「杵家彌七出版」** を創立 |
| 昭和2年 | 「杵屋」から「杵家」へ改名 |
| 以降 | 発行元が「三味線文化譜楽会」「株式会社邦楽社」へ一部移管 |
| 近年 | 六世杵家彌七改訂のA4版・B5版譜面が杵家彌七出版から発行される体制に回帰 |

### 出版された代表的楽曲

長唄の名曲群を中心とし、数十編に及ぶ選集が順次刊行されてきた。

- 『越後獅子』
- 『秋の色種』
- 『勧進帳』
- 『小鍛冶』
- 『汐汲』
- その他多数

### 現代的なアップデート

令和2年（2020年）には、手書きで譜面を修正したり行間に替手（対旋律）を書き込んだりできる **「三味線文化ふせん」** といった学習補助ツールも発売されるなど、現代の教育現場に合わせたアップデートが継続されている。

### 市販楽譜としての画期性

特筆すべきは、文化譜が単なる一部の流派の秘伝にとどまらず、**市販の楽譜として広く一般に流通** したことである。
これにより、師匠に直接師事しなくとも、楽譜を購入して独学で三味線の基礎を学ぶことが物理的に可能となり、
長唄のみならず他ジャンルへと越境する強力なプラットフォームを形成した。
